ABSTRACT

This tutorial focuses on differentially private (DP) synthetic data generation, addressing the need for robust anonymization in machine learning. Creating DP synthetic data allows for data sharing and analysis without compromising individuals' privacy, opening up possibilities for collaborative research and model training. The tutorial bridges gaps between DP training, inference, and empirical privacy testing, providing a comprehensive guide for generating DP synthetic data across tabular, image, and text data types.

We cover the full pipeline from introduction to synthetic data and differential privacy concepts, through specific generation methods for different data types, to practical system components including privacy guarantees, contribution constraints, and empirical privacy testing. The tutorial concludes with open questions and challenges in the field.
