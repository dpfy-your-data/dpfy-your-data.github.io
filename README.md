# DP-fy your DATA: ICML 2025 Tutorial Website

Official website for the ICML 2025 tutorial: **"DP-fy your DATA: How to (and why) synthesize Differentially Private Synthetic Data"**

## <¯ About

This tutorial focuses on differentially private (DP) synthetic data generation, addressing the need for robust anonymization in machine learning. We cover the full pipeline from introduction to synthetic data and differential privacy concepts, through specific generation methods for different data types, to practical system components including privacy guarantees, contribution constraints, and empirical privacy testing.
